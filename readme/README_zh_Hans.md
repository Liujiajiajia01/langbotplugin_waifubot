# Waifu情感机器人

一个拥有真实情感、记忆和人格系统的聊天机器人插件。

## 功能特性

- **情感系统**：能够感知用户的情感变化，建立好感度系统
- **记忆功能**：拥有短期记忆和长期记忆，能够记住与用户的对话内容
- **人格设定**：支持多种角色卡，可自定义角色性格和背景
- **拟人化表现**：支持打字时间、分段回复等拟人化特性
- **群聊支持**：可在群聊环境中与用户互动
- **主动交流**：能够主动发起对话，增加互动性

## 快速开始

1. 安装插件后，在插件设置中选择角色卡
2. 开启或关闭长期记忆、拟人化等功能
3. 开始与机器人对话，体验情感交互

## 核心命令

- `/affection` - 查看当前好感度
- `/affection reset` - 重置好感度
- `/memory` - 查看记忆内容
- `/memory clear` - 清除记忆

## 配置说明

插件提供了丰富的配置选项，包括：
- 角色卡选择
- 记忆系统设置
- 拟人化表现配置
- 群聊交互参数
- 主动发言设置
